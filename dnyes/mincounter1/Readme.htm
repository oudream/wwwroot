安装说明

  1. 将COCOON Counter 6复制到你的服务器上。
  2. 点击右上角的管理按钮，进入管理页面（初始密码为：admin）
  3. 在“基本设置”中，根据所给的参考，正确设置您的统计系统所在的URL。
  4. 修改管理密码。
  5. 点击"注册"按钮，注册新的统计用户。
  6. 获得统计代码，并将统计代码插入你需要统计的页面中。
  7. 访问http://你的统计路径/，用户名和注册时所填的密码登录查看统计情况。
